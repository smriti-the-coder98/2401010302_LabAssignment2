This project is part of Analysis and Design of Algorithms (ADA) Lab Assignment – 2. It demonstrates different algorithm design techniques and compares their performance using Python.

The assignment includes:

Task 1: Merge Sort using Divide & Conquer.
Task 2: Comparison of Bubble Sort, Merge Sort, and Quick Sort.
Task 3: Fractional Knapsack using Greedy approach.
Task 4: 0/1 Knapsack using Dynamic Programming.
Task 5: Travelling Salesman Problem (TSP) to observe exponential growth.

Execution time graphs are plotted to analyze and compare the efficiency of each algorithm.
